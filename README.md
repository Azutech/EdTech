# Bob's
This is a short description of a Bobs, testing his APIs. 


## Installation

To install the project, follow these steps:

-  Clone the repository: `git clone https://github.com/Azutech/EdTech`
-  Change to the project directory: `cd project`
-  Install the dependencies: `npm install`
-  Run the project: `npm start`

## Technologies

- Node.js: A JavaScript runtime environment.
- Express.js: A web application framework for Node.js.
- MongoDB: A NoSQL database for storing user and booking data.
- Mongoose: An Object Data Modeling (ODM) library for MongoDB.
- TypeScript: A typed superset of JavaScript.
